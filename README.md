This project is a sample implementation of [Recoil](https://recoiljs.org/).

## Getting Started

1. Clone the repository

   ```shell
   git clone https://github.com/joiellantero/react-recoil-demo.git
   ```

2. Install the dependencies

   ```shell
   npm install
   ```

3. Deploy locally

   ```shell
   npm run start
   ```
